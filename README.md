# 🪨📄✂️ Rock Paper Scissors Game (Python)

A simple command-line implementation of the classic **Rock, Paper, Scissors** game using Python.

---

## 🚀 Features

* Play against the computer 🤖
* Random choice generation for computer
* User input handling
* Clear win/lose/draw logic
* Beginner-friendly code

---

## 🛠️ Technologies Used

* Python 3

---

## 📂 Project Structure

```
.
├── rsp.py        # Main game file
└── README.md     # Project documentation
```

---

## ▶️ How to Run

### Using IDLE:

1. Open `rsp.py` in IDLE
2. Press **F5** or click **Run → Run Module**

### Using Terminal:

```bash
python3 rsp.py
```

---

## 🎮 How to Play

1. Run the program
2. Enter your choice:

   * `rock`
   * `paper`
   * `scissors`
3. The computer will randomly select its move
4. The result (Win/Lose/Tie) will be displayed

---

## 🧠 Game Rules

* Rock Smashes Scissors
* Scissors Cuts Paper
* Paper Covers Rock

---

## 📸 Example Output

```
Enter your choice (rock, paper, scissors): rock
Computer chose: scissors
Rock smashes scissors! You win! 🎉
```
